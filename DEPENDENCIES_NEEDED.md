## Charts Library

The enhanced funnel chart requires the `recharts` library.

### Installation

Run this command in your terminal:

```bash
npm install recharts
```

Or if using yarn:

```bash
yarn add recharts
```

### Type Definitions

TypeScript types are included with recharts, no additional @types package needed.

---

## Usage

After installation, the EnhancedFunnelChart component will work properly.

The chart visualizes:
- Total Leads
- Demo Showed
- Quotation Sent
- In Negotiation
- Converted

With percentages and color-coded bars.
